# ⚽ FC Barcelona Club Hub

A fan-focused single-page web application dedicated to **FC Barcelona**, showcasing the club’s history, identity, squad, fixtures, and legacy — built by [Harshvardhan Singh Karki](https://github.com/harshskarki).

🔗 **Live Demo:** [fc-barcelona-pi.vercel.app](https://fc-barcelona-pi.vercel.app/)

---

## 🏟️ Overview

**FC Barcelona Club Hub** is an interactive website that celebrates one of the most iconic football clubs in the world.  
It combines clean design, informative sections, and responsive UI to give users an engaging overview of the club’s spirit and achievements.

---

## ✨ Features

- 🏆 **About the Club:** Founding year, nicknames, colours, and background.  
- 🔵 **Identity & Philosophy:** Insights into La Masia, Catalan culture, and possession-based playstyle.  
- 🥇 **Major Honours:** A list of Barcelona’s most notable trophies.  
- 🌟 **Club Legends:** Tribute to iconic players from the club’s history.  
- 🏟️ **Stadium:** Info on the Camp Nou and its redevelopment.  
- 👥 **Current Squad:** Showcase of key players (demo data).  
- 📅 **Fixtures:** Sample upcoming match details.  
- 💻 **Responsive UI:** Works smoothly across all devices.

---

## 🧩 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Deployment:** [Vercel](https://vercel.com/)
- **Design:** Minimal and fan-centric static layout
